# Circuito-Digital
Termômetro Digital Simulado

Este projeto consiste em uma simulação de um termômetro digital, desenvolvido como parte da disciplina de Sistemas Digitais. O objetivo é demonstrar o funcionamento básico de um sistema digital para leitura, processamento e exibição de dados de temperatura.

bjetivo
O projeto tem como finalidade mostrar, por meio de simulação, como um sistema digital pode realizar a leitura de um sensor de temperatura, processar esses dados e apresentar o resultado em um display. Embora o sistema utilize valores simulados, ele reflete o funcionamento de um sistema real que poderia ser implementado com sensores físicos e microcontroladores.

Funcionamento do Sistema
O funcionamento do sistema é dividido em etapas que simulam as operações de um circuito digital:

Sensor de Temperatura (Simulado)
   Gera um valor de temperatura, representando a leitura de um sensor analógico.

   Conversão Analógico-Digital (ADC)
   O valor analógico é convertido para um valor digital, simulando a operação de um conversor ADC.

   Processamento do Circuito
   O valor digital é processado para calcular a temperatura final a ser exibida.
   
   Display Digital
   A temperatura processada é exibida em um display digital na interface do sistema.

Funcionalidades

* Simulação de sensor de temperatura.
* Conversão analógico-digital simulada.
* Exibição da temperatura em um display digital.
* Medição automática em tempo real.
* Atualização da temperatura a cada 1 segundo.
* Exibição do horário da última medição.
* Botões para iniciar e parar a medição.

Tecnologias Utilizadas

* HTML
* CSS
* JavaScript

Observação
Este projeto utiliza valores de temperatura simulados. Em uma implementação real, o sistema poderia ser conectado a um sensor de temperatura físico e a um microcontrolador para realizar medições reais do ambiente.
